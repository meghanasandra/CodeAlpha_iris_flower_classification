# Iris Flower Classification

## Objective
Classify Iris flowers using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Machine Learning Algorithm
K-Nearest Neighbors (KNN)

## Accuracy
Achieved accuracy around 95%–100%.

## Dataset
Iris flower dataset containing:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Output
The model predicts:
- Iris-setosa
- Iris-versicolor
- Iris-virginica
